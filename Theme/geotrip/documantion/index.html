توضیحات

با دنبال کردن این راهنما می توانید سفارشی سازی قالب GeoTrip را شروع کنید، این راهنما به شما کمک می کند تا به سرعت و با درک آسان بیشترین استفاده را از قالب HTML ما ببرید.

از شما برای خرید قالب GeoTrip سپاسگزاریم! قبل از شروع سفارشی سازی آن ، توصیه می شود مراحل ذکر شده در این راهنما را مطالعه کنید. ما اطلاعات لازم را ذکر کرده ایم و تمام دستورالعمل‌ها و جزئیاتی را که برای استفاده از این قالب نیاز دارید در اختیار شما قرار داده ایم.

ساختار پوشه ها

پس از دانلود و Extract پوشه، ساختار فایل ها و پوشه های به صورت زیر خواهد بود :

  • نسخه ltr و rtl
    شامل همه فایل های نسخه اصلی (LTR) و نسخه راست چین شده ی (RTL) قالب می باشد
      documantion
      شامل فایل‌های راهنما جهت ویرایش و سفارشی سازی قالب
        index.html
        صفحه توضیحات
        assets
        شامل فایل‌های CSS، Images، JS و افزونه‌ها
        • CSS
          شامل فایل های Css
        • webfonts
          شامل فایل های آیکُن فونت ها و فونت های فارسی
        • img
          شامل تمام تصاویر استفاده شده در فایل های html
        • js
          شامل فایل های جاوااسکریپت

اقدامات لازم بعد از خرید

1. بررسی فایل ها

بعد از خرید قالب از سایت راست چین فایل .zip را extract کنید

2. سفارشی سازی

اکنون می توانید سفارشی سازی وب سایت خود را بر اساس نیاز خود شروع کنید.

3. حذف فایل و پوشه های غیرضروری

پس از تکمیل وب سایت خود، باید فایل و پوشه غیر ضروری را حذف کنید تا حجم وب سایت کاهش یابد.
