فونت آیکُن

ما در قالب GeoTrip از Font awesome و Bootstrap Icons استفاده کرده ایم.

Font awesome:

از این آیکن ها، باید از یک پیشوند fa- با نام آیکن برای مثال camera استفاده کنید.


				<i class="fa-brands fa-facebook"></i> <!-- this icon's 1) style prefix == fa-solid and 2) icon name == camera -->
				<i class="fa-solid fa-user"></i> <!-- using an <i> element to reference the icon -->
				<span class="fa-solid fa-heart"></span> <!-- using a <span> element to reference the icon -->

لیستی کامل از آیکن های Font awesome را می توانید در سایت رسمی Font awesome مشاهده کنید.
